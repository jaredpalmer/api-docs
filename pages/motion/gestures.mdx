# Gestures

<span className="lead">
  A powerful gesture recognition system for the browser.
</span>

<div>

Framer Motion extends the basic set of gestures provided by React with a simple yet powerful set of UI gesture recognisers.

It also provides `whileHover` and `whileTap` props that make it easy to temporarily animate to a state while a gesture is active.

</div>

```jsx
import { motion } from "framer-motion"

export function MyComponent() {
  return (
    <motion.div
      drag
      whileHover={{ scale: 1.2 }}
      whileTap={{ scale: 0.9 }}
    />
}
```

## Tap

A tap is a convenience alternative to the native browser `click` event.

- It works on both mobile and desktop, leveraging both mouse and touch events depending on the platform.
- It only fires if `up` was received on the same `motion` component that received the initial `down` to mimic interface button behaviour.
- It does not fire if panning has been initiated on this or any other component as a result of the same pointer event.
